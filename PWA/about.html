Progressive Web App

A Progressive Web Apps is a web app that uses modern technologies to make an app like experience. They are, reliable and more engaging. Just like the advantages, PWAs have some additional requirements to be accepted by web browsers.
These requirements are:

Secure:

A secure connection (HTTPS) over your site makes sure all traffic is as safe as a native app. Service Workers cannot be registered on for a webapp if it is not hosted on HTTPS.


Service Worker:

The webapp should have a valid servieworker.js file. It can cache data and receive background notifications, which are the main specialities of PWA.


Manifest.json:

The webapp should have a valid manifest.json file with details about the behaviour of that app. This includes everything from images, to language, to the start page of your web app.


Network Independant:

Progressive Web App should work properly without depending on network connection. Service worker will help PWA to control tracfic even if client is offline or the network isn't reliable.


Reponsiveness:

The site should be responsive on all screen sizes like Mobile, Tablet and Desktop.


Cross-Browser:

The PWA should work properly on multiple web browser (Like Chrome, Firefox, Edge and Safari) without causing any kind of errors.
